# 🧱 BrickBazaar

A full-stack MERN marketplace for construction materials connecting buyers and sellers.

## 🚀 Features

- 🔐 JWT Authentication with role-based access (Buyer/Seller)
- 📦 Product Management (CRUD operations)
- 🛒 Order System with real-time tracking
- 📊 Personalized Dashboards
- 📸 Image Upload functionality
- 🎨 Responsive UI/UX

## 💻 Tech Stack

**Frontend:**
- React.js
- React Router
- Axios
- Context API
- CSS3

**Backend:**
- Node.js
- Express.js
- MongoDB
- JWT
- Multer
